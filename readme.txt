Треп Хата — static site
Вміст:
- index.html (Головна)
- events.html (Події)
- vip.html (VIP — Усадьба)
- contacts.html (Контакти)
- styles.css (стилі)
- script.js (меню)
Інструкції:
1) Розпакуйте папку та завантажте файли на ваш хостинг (FTP / File Manager) або відкрийте index.html локально в браузері.
2) Щоб опублікувати без хостингу, можна використати GitHub Pages: створіть репозиторій, додайте файли у root та увімкніть Pages.
3) Щоб змінити телефон або тексти — відкрийте файли .html у будь-якому текстовому редакторі.
